# **App Name**: Registre Diari de Treball

## Core Features:

- Identificació d'Empleat: Permet als empleats identificar-se de forma segura mitjançant credencials d'usuari úniques.
- Registre de Servei: Permet als empleats registrar nous serveis amb camps per a la descripció.
- Seguiment d'Hora: Registre les hores d'arribada i sortida per a cada servei.
- Descripció del Servei: Permet als empleats proporcionar una descripció detallada de les tasques realitzades.
- Captura de Fotos: Permet als empleats capturar i desar fotos relacionades amb el servei.
- Generació d'Informe Diari: Genera un informe diari que resumeix els serveis registrats, les hores i les fotos.
- Emmagatzematge d'Informació: Guarda tota la informació en una base de dades.

## Style Guidelines:

- Color primari: Blau tranquil (#64B5F6) per a la fiabilitat i l'eficiència.
- Color de fons: Gris clar (#F5F5F5) per a una interfície neta.
- Color d'accent: Taronja càlid (#FFB74D) per destacar les accions importants.
- Font per al cos del text i titulars: 'Inter', una sans-serif moderna, per a una bona llegibilitat.
- Utilitzar icones clares i representatives per a les diferents funcions (registre, horari, descripció, foto).
- Disseny net i ordenat per a facilitar l'entrada de dades i la revisió dels registres.
- Animacions subtils per indicar les interaccions i carregar les operacions (per exemple, transicions suaus entre pantalles).
